This project predicts employee attrition using IBM HR Analytics dataset and provides actionable insights for HR management. It combines **Python-based analytics**, **machine learning modeling**, and **Power BI visualizations**.

##  Objectives
- Predict which employees are likely to leave.
- Identify key factors influencing attrition.
- Provide a visual dashboard for HR decision-making.

##  Tools & Technologies
- Python: Pandas, Seaborn, Scikit-learn, Matplotlib
- Power BI Desktop
- Jupyter Notebook

##  Steps
1. **Data Collection**: Import dataset from Kaggle.
2. **Data Cleaning**: Handle missing values, encode categorical variables.
3. **Exploratory Data Analysis (EDA)**: Visualize trends and correlations.
4. **Model Building**: Logistic Regression & Decision Tree.
5. **Model Explainability**: Identify important factors (e.g., Job Satisfaction, Monthly Income).
6. **Power BI Dashboard**: Interactive visualizations of attrition trends.
7. **Insights & Recommendations**: Actionable strategies to reduce attrition.
